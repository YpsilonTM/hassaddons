# ModQTT

ModQTT is a general-purpose Modbus TCP to MQTT bridge for Home Assistant.

It reads registers from any Modbus TCP device (inverters, meters, sensors, or industrial equipment), decodes values with configurable endianness and scaling, and publishes clean MQTT state topics that Home Assistant can consume.
